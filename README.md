# עלון שבת — בית מנחם

אתר סטטי (GitHub Pages) ליצירת עלון שבת שבועי לבית כנסת **"בית מנחם" חב״ד גבעת שרת — בית שמש**.

## כתובת חיה

**https://shneurgreenberg.github.io/beit-menachem-bulletin/**

> שימו לב: `https://shneurgreenberg.github.io/` הוא ריפו נפרד (`shneurGreenberg.github.io`) עם אפליקציית Flutter ישנה משנת 2022 שמציגה מסך ריק. העלון עצמו נמצא בנתיב למעלה. כדי שהשורש יפנה לעלון, הריצו:
>
> ```bash
> node scripts/fix-user-pages.mjs
> ```
>
> (דורש הרשאת כתיבה לריפו `shneurGreenberg.github.io`). הקבצים להעתקה ידנית נמצאים ב־`user-site/`.

## מה כלול

- חלוקה ברורה: **שבת נוכחית** מול **השבוע הקרוב** (כולל שם הפרשה הבאה)
- עיצוב שחור־לבן־אפור להדפסה + הודעה חשובה בצבע שבת
- חישוב זמנים אוטומטי לפי בית שמש (Hebcal) עם יחסי מנחה/שקיעה מהעלון
- עיגול מנחה/ערבית בימי חול לשעות "נורמליות" (`:00` / `:10` / `:15` / `:30` / `:45`)
- מצב עריכה עם סיסמה, שינוי ידני, ואיפוס לזמנים אוטומטיים
- היסטוריית פרשות, תבניות הודעה, שיתוף WhatsApp, הדפסה/PDF
- GitHub Action לרענון שבועי של `data/week.json` (גיבוי)

## סיסמת עריכה

ברירת מחדל: `menachem`  
(מוגדרת ב־`js/config.js`)

## פיתוח מקומי

אין צורך ב־build. מספיק להגיש את התיקייה כשרת סטטי:

```bash
npx --yes serve .
```

ואז לפתוח את הכתובת שמוצגת בטרמינל.

## טבלת זמנים (אדמו״ר הזקן — בית שמש)

המקור העיקרי לזמני השקיעה/צאת הוא `data/zmanim-table.json`, שנגזר מקובץ האקסל
`data/beit-shemesh-5786.xlsx` (ה׳תשפ״ו).

לרענון אחרי החלפת קובץ האקסל:

```bash
node scripts/seed-zmanim-table.mjs
```

(דורש ספריית `xlsx` — ראו הערות בראש הסקריפט.)

Hebcal משמש לפרשה/תאריך עברי ולגיבוי אם אין שורה ליום בטבלה.
