# עלון שבת — בית מנחם

אתר סטטי (GitHub Pages) ליצירת עלון שבת שבועי לבית כנסת **"בית מנחם" חב״ד גבעת שרת — בית שמש**.

## מה כלול

- חלוקה ברורה: **שבת נוכחית** מול **השבוע הקרוב** (כולל שם הפרשה הבאה)
- עיצוב שחור־לבן־אפור להדפסה + הודעה חשובה בצבע שבת
- חישוב זמנים אוטומטי לפי בית שמש (Hebcal) עם יחסי מנחה/שקיעה מהעלון
- עיגול מנחה/ערבית בימי חול לשעות "נורמליות" (`:00` / `:10` / `:15` / `:30` / `:45`)
- מצב עריכה עם סיסמה, שינוי ידני, ואיפוס לזמנים אוטומטיים
- היסטוריית פרשות, תבניות הודעה, שיתוף WhatsApp, הדפסה/PDF
- GitHub Action לרענון שבועי של `data/week.json` (גיבוי)

## סיסמת עריכה

ברירת מחדל: `menachem`  
(מוגדרת ב־`js/config.js`)

## פיתוח מקומי

אין צורך ב־build. מספיק להגיש את התיקייה כשרת סטטי:

```bash
npx --yes serve .
```

ואז לפתוח את הכתובת שמוצגת בטרמינל.

## עדכון טבלת זמנים מקומית

כרגע המקור הוא Hebcal לבית שמש. כשתגיע הטבלה שלכם — נעדכן את `js/times.js` / `js/config.js` כך שהשקיעה והזמנים ייגזרו מהטבלה במקום מה־API.

## GitHub Pages

האתר מוגש משורש הריפו (`index.html`). לאחר הפעלה ב־Settings → Pages → Deploy from branch `main` / root.
